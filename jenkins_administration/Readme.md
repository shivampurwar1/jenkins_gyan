## Jenkins Console
### safeExit
<jenkins-url>/safeExit
All jobs will complete then server gracefully exit for maintenance.

### safeRestart
<jenkins-url>/restart 
This prompt for graceful restart.

### restart
<jenkins-url>/restart 
This prompt for force restart (not graceful)
